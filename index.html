Rule Firewall untuk Allow DNS (Redirect)

                
                
# -- Metode Redirect (Paling Direkomendasikan) --
# Memaksa semua klien menggunakan DNS dari router untuk cache dan kontrol.

# Langkah 1: Alihkan semua permintaan DNS ke router
/ip firewall nat
add action=redirect chain=dstnat comment="Redirect DNS ke Router" dst-port=53 protocol=tcp to-ports=53
add action=redirect chain=dstnat comment="Redirect DNS ke Router" dst-port=53 protocol=udp to-ports=53

# Langkah 2: Izinkan router menerima permintaan DNS dari jaringan lokal
/ip dns
set allow-remote-requests=yes
